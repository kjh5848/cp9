# 마크다운 텍스트 강조 스페이싱 룰 (Markdown Bold Spacing Rule)
마크다운 볼드체(`**`)를 사용하여 텍스트를 강조할 때, 텍스트(괄호) `텍스트() `, 따옴표 `"" ` 문구 뒤에는 반드시 공백을 넣어야 합니다. (일반 텍스트 제외)
- 올바른 예시: `**한글(영어)** 을`, `**"강조"** 함`,`**"문서(Vector)에서 찾을 것인가?"** 를`
- 잘못된 예시: `**한글 (영어)**을`, `**"강조"**함`,`**"문서(Vector)에서 찾을 것인가?"**를`
